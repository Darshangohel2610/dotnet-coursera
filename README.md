# Coursera Final Assignment - ASP.NET Core API

## Features
- Full CRUD API (Create, Read, Update, Delete)
- Clean Architecture (Controller → Service → Model)
- Custom Middleware:
  - Exception Handling
  - Request Timing
- Swagger UI for testing

## Endpoints
- GET /api/products
- GET /api/products/{id}
- POST /api/products
- PUT /api/products/{id}
- DELETE /api/products/{id}

## Run Project
1. dotnet restore
2. dotnet run
3. Open http://localhost:5000/swagger
